A. index.html
Este será el primer archivo que crearemos.

¿Qué hace?: Define los bloques de contenido (Hero, Servicios, Galería, Contacto).

Prioridad: Aquí estableceremos los "IDs" para que los botones de WhatsApp e Instagram que mencionaste en el PDF funcionen perfectamente al hacer clic.

B. style.css
¿Qué hace?: Aquí inyectaremos la paleta de colores del PDF (#161212, #9C8D7C, etc.).

Prioridad: Configuraremos el sistema de "Grillas" (CSS Grid) para que las fotos de parejas, maternidad y deportes se vean impecables tanto en computadoras como en celulares (Responsive Design).

C. script.js
¿Qué hace?: Controlará efectos sutiles, como que el menú cambie de color cuando el usuario baje la página o animaciones de entrada para las fotos.

Prioridad: Asegurar que la experiencia de navegación sea fluida y "premium".
